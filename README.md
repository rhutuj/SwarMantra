# Swar Notebook V1

Swar Notebook is a desktop app for Hindustani Classical Music students and teachers.

## Stack

- React
- TypeScript
- Tauri
- SQLite
- Prisma
- Zustand
- Tailwind CSS

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Generate Prisma client:
   ```bash
   npm run prisma:generate
   ```
3. Start the app:
   ```bash
   npm run dev
   ```
